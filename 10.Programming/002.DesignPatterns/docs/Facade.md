# Facade Pattern

> Structural Pattern

> **Gof**
>
> Provide a unified interface to a set of interfaces in a subsystem. `Facade` defines a higher-level interface that makes the subsystem easier to use.

> **Refactoring guru**
>
> **Facade** is a structural design pattern that provides a simplified interface to a library, a framework, or any other complex set of classes.

The `Facade pattern` provides an external interface for use, but it does not introduce additional logic; it merely coordinates and executes specific subsystem functions in sequence.

This reduces coupling so that clients are only coupled with the Facade class.
